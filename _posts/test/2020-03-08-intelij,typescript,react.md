IntelliJ에서 react typescript 프로젝트 만들기!
```
1.  create New project
2.  static web 
3. react app
4. 프로젝트 명 입력
5. create typescript project
```

실행
```
1. terminal에서 아래를 입력한다.
$> npm start
2. localhost:3000으로 기본페이지가 뜬다.
```
